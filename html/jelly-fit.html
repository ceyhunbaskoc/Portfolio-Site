🟢 Jelly Fit (Unity – UDO Games Summer Internship 2025)

[Unity | UDO Games Summer Internship 2025]

Genre: 3D Puzzle / Logic Game
Status: Completed Internship Project

🎮 About the Game

Jelly Fit is a 3D puzzle game designed to challenge the player’s logical reasoning and spatial awareness.
The goal is to fit differently shaped cube groups into a confined target box without spilling or leaving empty spaces.
Players can drag, drop, and even cut cubes when necessary to complete the puzzle successfully.

The game emphasizes strategic planning and problem-solving, making each level a fun and rewarding experience.

🌟 Key Features

🟢 Drag & Drop Mechanics

  • Select and move cube groups intuitively.
  • Rotate or position cubes to fit them inside the target box.

✂️ Cutting Mechanics

  • Split cube groups to better match the shape of the target box.
  • Allows creative solutions for complex puzzles.

🧩 Logical Puzzle Design

  • Levels are designed to increase in complexity gradually.
  • Encourages players to think ahead and plan moves strategically.

🔲 Grid System

  • The grid system ensures precise placement of cubes.
  • Tracks filled and empty cells for accurate puzzle completion.

🎵 Audio & Animations

  • Sound manager handles feedback for dragging, dropping, and cutting.
  • Smooth animations enhance visual feedback and gameplay satisfaction.

🕹️ Gameplay Loop

  1. Observe the target box shape.
  2. Drag and drop cube groups into the box.
  3. Cut or rotate cubes as necessary to fill the box completely.
  4. Complete the puzzle without spilling cubes.
  5. Advance to more challenging levels with larger grids or irregular shapes.

🛠️ Technical Highlights

  • Built entirely in Unity using C#.
  • Grid system ensures perfect alignment and placement detection.
  • Drag & drop mechanics include collision detection and smooth snapping.
  • Cutting mechanic dynamically splits cube groups into smaller units.
  • Fully modular sound manager supporting multiple feedback cues.
  • Optimized for both desktop and mobile platforms.

🧩 Design Philosophy

Jelly Fit focuses on cognitive satisfaction:

  • Immediate visual and audio feedback for every action.
  • Encourages logical thinking and experimentation.
  • Each puzzle is solvable but requires careful planning.
